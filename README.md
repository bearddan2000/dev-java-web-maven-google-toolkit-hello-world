# dev-java-web-maven-google-toolkit-hello-world

## Description
A POC for gwt framework rendering html page.

Opens a developer's window for jetty so we have
a desktop/web hybrid framework.

## Tech stack
- rpc
- swt
- gwt
- jetty
- chrome

## Docker stack
- maven:3-openjdk-8

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
